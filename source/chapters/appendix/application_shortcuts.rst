Application shortcuts
=====================

Use these shortcuts to access features available in the menu on top of the
application window.

========================================  ================================
Function                                  Key
========================================  ================================
**File menu**
--------------------------------------------------------------------------
Load Track to Deck 1                      :kbd:`Ctrl` + :kbd:`O`
----------------------------------------  --------------------------------
Load Track to Deck 2                      :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`O`
----------------------------------------  --------------------------------
**Library menu**
--------------------------------------------------------------------------
Add new Playlist                          :kbd:`Ctrl` + :kbd:`N`
----------------------------------------  --------------------------------
Add new Crate                             :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`N`
----------------------------------------  --------------------------------
**View menu**
--------------------------------------------------------------------------
Show Skin Settings Menu                   :kbd:`Ctrl` + :kbd:`1`
----------------------------------------  --------------------------------
Show Microphone Section                   :kbd:`Ctrl` + :kbd:`2`
----------------------------------------  --------------------------------
Show Vinyl Control Section                :kbd:`Ctrl` + :kbd:`3`
----------------------------------------  --------------------------------
Show Preview Deck                         :kbd:`Ctrl` + :kbd:`4`
----------------------------------------  --------------------------------
Show Cover Art                            :kbd:`Ctrl` + :kbd:`6`
----------------------------------------  --------------------------------
Maximize Library                          :kbd:`Space`
----------------------------------------  --------------------------------
Full Screen (Windows & Linux)             :kbd:`F11`
----------------------------------------  --------------------------------
Full Screen (macOS)                       :kbd:`Shift` + :kbd:`Command` + :kbd:`F`
----------------------------------------  --------------------------------
**Options menu**
--------------------------------------------------------------------------
Enable Vinyl Control 1                    :kbd:`Ctrl` + :kbd:`T`
----------------------------------------  --------------------------------
Enable Vinyl Control 2                    :kbd:`Ctrl` + :kbd:`Y`
----------------------------------------  --------------------------------
Enable Vinyl Control 3                    :kbd:`Ctrl` + :kbd:`U`
----------------------------------------  --------------------------------
Enable Vinyl Control 4                    :kbd:`Ctrl` + :kbd:`I`
----------------------------------------  --------------------------------
Enable Live Broadcasting                  :kbd:`Ctrl` + :kbd:`L`
----------------------------------------  --------------------------------
Record Mix                                :kbd:`Ctrl` + :kbd:`R`
----------------------------------------  --------------------------------
Enable Keyboard Shortcuts                 :kbd:`Ctrl` + :kbd:`\``
----------------------------------------  --------------------------------
Open Preferences (Windows & Linux)        :kbd:`Ctrl` + :kbd:`P`
----------------------------------------  --------------------------------
Open Preferences (macOS)                  :kbd:`Ctrl` + :kbd:`,`
----------------------------------------  --------------------------------
Exit Mixxx                                :kbd:`Ctrl` + :kbd:`Q`
----------------------------------------  --------------------------------
**Developer menu**
--------------------------------------------------------------------------
Reload skin                               :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`R`
----------------------------------------  --------------------------------
Developer Tools                           :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`T`
----------------------------------------  --------------------------------
Stats: Base Bucket                        :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`B`
----------------------------------------  --------------------------------
Stats: Experiment Bucket                  :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`E`
----------------------------------------  --------------------------------
Debugger Enabled                          :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`D`
========================================  ================================

.. hint:: The :kbd:`Ctrl` key on Windows & Linux is equivalent to the
          :kbd:`Command` key on macOS. The :kbd:`Alt` key on Windows & Linux
          is equivalent to the :kbd:`Option` key on macOS. Application
          shortcuts are not customizable.

.. note:: The Developer menu is available if Mixxx is started using the
          :ref:`appendix-command-line-options`.
