This device is not USB MIDI class compliant. Its signals are translated to MIDI by special drivers on Windows and Mac OS X. There is no driver available for Linux. It has been discontinued in favor of the {ref}`Novation Launchpad MK2 <novation-launchpad-mk2>` which is USB MIDI class compliant and has RGB pads.
