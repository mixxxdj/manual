# Novation Dicer

<!-- Missing product shot image: novation_dicer_product_shot.jpg -->

- [Manufacturer’s product page](https://web.archive.org/web/20120605072402/http://www.novationmusic.com/global/products/digital_dj/dicer/) (archived)

The Novation Dicer is a small controller designed to be placed on the corner of turntables for use with timecode vinyl. They are sold in pairs.

:::{versionadded} 1.10
:::

## Mapping description

**Author:** Shaun (DJ Pandemonium)

The mapping has all buttons mapped and mimics the overall layout of the [DJ Tech Tools Dicer mapping for Traktor](http://www.djtechtools.com/2010/11/11/novation-dicer-mapping-for-traktor-scratch/).

### Press Dice buttons 1-5 whilst holding down Mode button ([R]ed/[G]reen/[O]range) to access the shifted layer functions.

| Button | Standard layer | | | | | Shift layer | | |
| ------ | -------------- |-|-|-|-| ----------- |-|-|-|
| | [R] | [G] | [O] | | | [R] | [G] | [O] |
| Btn 2 | Set/Play Hotcue 1 | Loop Roll 1/16 beat | Auto Loop 1 Beat | | Btn 2 | Delete Hotcue 1 | Forward then Resume | Next Playlist |
| Btn 1 | Set/Play Hotcue 2 | Loop Roll 1/8 beat | Auto Loop 2 Beat | | Btn 1 | Delete Hotcue 2 | Cue | Next Track |
| Btn 3 | Set/Play Hotcue 3 | Loop Roll 1/4 beat | Auto Loop 4 Beat | | Btn 3 | Delete Hotcue 3 | Toggle Flanger | Load Track |
| Btn 4 | Set/Play Hotcue 4 | Loop Roll 1/2 beat | Auto Loop 8 Beat | | Btn 4 | Delete Hotcue 4 | Transform | Previous Track |
| Btn 5 | Set/Play Hotcue 5 | Loop Roll 1 beat | Auto Loop 16 Beat | | Btn 5 | Delete Hotcue 5 | Rewind then Resume | Previous Playlist |
