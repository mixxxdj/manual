# Livingstone MID-100

:::{note}
Unfortunately a detailed description of this controller mapping is still missing.
If you own this controller, please consider
[contributing one](https://github.com/mixxxdj/mixxx/wiki/Contributing-Mappings#user-content-documenting-the-mapping).
:::

The Livingstone MID-100 is a variant of the :ref:`ION Discover DJ <ion-discover-dj-pro>`
USB MIDI controller. It features a class compliant USB interface that
does not require drivers. The MID-100 does not provide a built-in sound
card.
